## Description

Unity 2D inverted shoot-em-up.

**In this game, you command a powerful fleet of ships to face off against the ultimate solo pilot who defies gravity and common sense.**

### Controls

* Battlecruiser attack: Z.
* Frigate attack: X.
* Bomber shield activation: C.
* Control the position of the fleet by moving the cursor to the right or left edge of the screen.
* Control the compactness with the mouse wheel.

### Enemy

But your enemy is also not an easy one. He has three attack modes:
* basic
* master of evasion
* devil

## Getting Started

### Installing

* Windows version: [link](https://github.com/Severnbli/Shoot-Em-Up/releases/download/stable/agalaG.zip).

### Building

1. Download project via `git clone` (preferred) or [as .zip file](https://github.com/Severnbli/Shoot-Em-Up/archive/refs/heads/main.zip).
2. Go to Unity and open project. Project unity version is `2022.3.50f1`.
3. In Editor go to File → Build Profiles.
4. Build project.
